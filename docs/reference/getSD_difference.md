# getSD_difference

Estimate standard deviation of difference in means before and after
change for paired t-test of difference in means.

## Usage

``` r
getSD_difference(sd_within, nA, nB, sd_delta)
```

## Arguments

- sd_within:

  standard deviation within sites

- nA:

  number of after samples per site

- nB:

  number of before samples per site

- sd_delta:

  standard deviation of true changes among sites

## Value

Standard deviation of the difference in means
