# getSD_within

Estimate within-site standard deviation from baseline data.

## Usage

``` r
getSD_within(baseline, siteVar = "site", responseVar = "y")
```

## Arguments

- baseline:

  Data frame containing baseline measurements

- siteVar:

  Name of the site variable in baseline data

- responseVar:

  Name of the response variable in baseline data

## Value

Estimated within-site standard deviation
